# v1.0.0 WeFlow 微信聊天导出套件

## 包含内容

- WeFlow 4.5.1 x64 安装器。
- `weflow-export-wechat` Codex Skill。
- ChatLab JSONL 与媒体相对路径验收工具。
- 面向同事的自包含 HTML 截图教程和精简启动提示词作为独立 Release 附件分发，不放入 `WeFlow.zip`。
- Mac / ExportWeChat 流程文档：安装、格式限制、导出目录验收、增量状态和脱敏处理。

## 已验证

- Skill 结构通过官方校验器。
- 真实导出样本：507条消息、137个成员记录、79个媒体引用，全部通过。
- 自动化增量样本：2605群当天增量从触发到任务中心完成约8分钟，JSONL中51条消息、16个图片引用通过完整性检查。
- 故障测试：能够发现损坏 JSONL、缺失媒体和路径越界。
- `WeFlow.zip` 不包含同事版 HTML 教程，只保留 Codex 下载所需工具包。
- HTML 教程已内嵌截图，单独下载也能看到配图。
- Release 只保留最少必要附件：`WeFlow.html`、`Codex.md`、`WeFlow.zip`、`SHA256SUMS.txt`。
- `Codex.md` 已压缩为短入口，详细流程由 Codex 阅读仓库文档和 Skill。

## 增量导出建议

- 已完成首次配置后，日常更新优先选“今天”或“从上次导出日期到今天”，不要默认全量导出。
- 每个群聊保留 `export_integrity_report.md`，用末条时间作为下一次增量起点。
- WeFlow 若只能按日期选择，可多导一天，再由 Codex 按消息ID、时间和内容结构去重。
- 批量排期参考：当天增量约2–5分钟/群，媒体较多约5–15分钟/群。

## 安全提示

- 安装器无数字签名，下载后必须核对 Release 公布的 SHA-256。
- WeFlow 为第三方工具；请在获得授权、合规且必要的范围内使用。
- 不要上传微信密钥、数据库或真实聊天导出包。
