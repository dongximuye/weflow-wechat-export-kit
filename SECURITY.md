# 安全与隐私说明

## 聊天数据

- 微信聊天记录、数据库、密钥和真实导出包仅在用户电脑本地处理。
- 不要把真实 JSONL、图片、文件、语音或视频提交到 GitHub。
- 分享分析结果前检查姓名、手机号、二维码、合同和其他敏感内容。

## 解密密钥

- Codex不需要看到密钥。
- 不复制、不截图、不发送、不写入日志或文档。
- 获取密钥时严格按照 WeFlow 界面提示，由用户本人确认退出或重新打开微信。

## WeFlow 安装器

- 安装器属于第三方软件。
- 发布前确认来源、版本、数字签名状态、SHA-256 和再分发权限。
- 安装器不进入 Git 历史，只能作为受控 Release 附件分发。
- 同事下载后应核对 Release 中公布的 SHA-256。

## 同事版教程

- HTML 教程只放操作截图和流程说明，不放真实聊天记录、数据库、密钥或导出样本。
- 截图如包含 wxid、群名、联系人、头像或账号信息，发布前必须打码。
- 教程和启动提示词可以单独发给同事；`WeFlow.zip` 仍只作为 Codex 下载的工具包。
